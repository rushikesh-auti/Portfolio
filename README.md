# Personal Portfolio Website

A modern and responsive personal portfolio website designed to showcase my skills, projects, and contact information. This portfolio reflects my journey as a Computer Science graduate and highlights my technical expertise and hands-on project experience.

##  Features

- Responsive design for desktop, tablet, and mobile devices
- Interactive navigation bar with hamburger menu
- Smooth scrolling and section highlighting
- Skills section with progress indicators
- Projects showcase with GitHub links
- Contact section with clickable social and email icons
- Clean UI with modern color combinations

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
