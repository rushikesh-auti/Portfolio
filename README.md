# Personal Portfolio Website

A modern, responsive personal portfolio website developed to showcase my skills, projects, and contact information.  
This project highlights my front-end development abilities, design approach, and practical experience with web technologies.

---

## Live Demo

https://portfoliorushiz.vercel.app/

---

## Overview

This portfolio website serves as a central place to present my technical skills, featured projects, and professional background.  
The design focuses on clarity, responsiveness, and user-friendly navigation.

---

## Key Features

- Fully responsive layout for all screen sizes
- Interactive navigation with hamburger menu
- Smooth scrolling and active section highlighting
- Skills section with visual indicators
- Projects section with GitHub repository links
- Contact section with social and email integration
- Clean, modern, and minimal UI design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts
